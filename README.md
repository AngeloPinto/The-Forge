# The-Forge

![The Forge](./img/thor-hammer_128x128.png)

**The Forge** É um aplicativo desenvolvido em [AutoHotkey](https://www.autohotkey.com) que tem como objetivo auxiliar desenvolvedores, com diversas ferramentas, a forjar seus softwares.

<br>

## Experimente

**No seu terminal ou prompt de comando:**

1. <kbd>gitlp</kbd> - Digite o comando e pressione <kbd>Enter</kbd> ou <kbd>Espaço</kbd>

2. <kbd>gitlpf</kbd> - Digite o comando e pressione <kbd>Enter</kbd> ou <kbd>Espaço</kbd>

3. <kbd>gitlgo</kbd> - Digite o comando e pressione <kbd>Enter</kbd> ou <kbd>Espaço</kbd>
